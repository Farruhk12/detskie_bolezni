# Образовательная платформа КДБ №1

## Структура файлов

| Файл | Назначение |
|------|------------|
| `index.html` | Главная страница для студентов |
| `admin.html` | Главная админки (навигация) |
| `admin-topics.html` | Добавление и редактирование тем |
| `admin-exam.html` | Добавление экзаменационных вопросов |
| `admin-topics.js` | Логика тем |
| `admin-exam.js` | Логика вопросов |
| `styles.css` | Общие стили |
| `config.js` | URL Google Apps Script и сервера ИИ |
| `app.js` | Логика основного приложения |
| `server.js` | Локальный сервер с ИИ (OpenAI) |
| `code.gs` | Скрипт для Google Таблицы |

## Запуск

### 1. Сайт (без Node.js сервера)
Откройте `index.html` в браузере или разверните на Vercel/Netlify.  
ИИ (чат, карточки, «Проверить себя») работает через Google Apps Script — **сервер не нужен**.

### 2. Настройка ИИ в Google Apps Script
В проекте GAS добавьте свойство:
- **GEMINI_API_KEY** — ключ из [Google AI Studio](https://aistudio.google.com/apikey)
- **GEMINI_MODEL** (опционально) — по умолчанию `gemma-3-27b-it` (выше лимит RPM на Free tier, чем у Flash)

Проект → Свойства скрипта → Добавить свойство → `GEMINI_API_KEY` = ваш ключ

### 3. (Опционально) Локальный сервер
Для разработки: `node server.js` — тогда ИИ идёт через Node вместо GAS.

### 4. Админ-панель
Откройте `admin.html` — там ссылки на два раздела:
- **Добавление тем** (`admin-topics.html`) — создание и редактирование тем
- **Экзаменационные вопросы** (`admin-exam.html`) — добавление вопросов по теме

Пароль задаётся в Google Apps Script (ADMIN_PASSWORD).

## Экзаменационные вопросы

В админке: вставьте текст из Word. Поддерживается формат:
- **@5Вопрос** — каждая строка начинается с @5, далее текст вопроса
- `1. Вопрос` или `1) Вопрос` — альтернативный формат

Это просто список вопросов, не тесты с вариантами ответов.

## Google Apps Script

Скопируйте `code.gs` в ваш проект GAS. Настройте:
- **ADMIN_PASSWORD** — пароль админ-панели
- **GEMINI_API_KEY** — ключ Gemini для ИИ (чат, карточки, оценка)
- **GEMINI_MODEL** — опционально (см. выше)
- URL развёртывания укажите в `config.js` (GAS_API)
